"Hallo Welt!"-Programm

Artikel erstellt: 26.09.2024 • Artikel zuletzt überarbeitet: 26.09.2024 • 0 mal überarbeitet

Ein "Hallo Welt!"-Programm gibt Text auf der Konsole aus. Mit dem Befehl Console.WriteLine() kann man Text in einer neuen Zeile ausgeben.

"Hallo Welt!"-Programm

Kopieren

// simples "Hallo Welt!"-Programm


using System;


class Program

{

  static void Main(string[] args)

  {

    Console.WriteLine("Hallo Welt!");

  }

}

Soll der Text in der gleichen Zeile erscheinen, verwendet man Console.Write().

Beispiel: Text in einer Zeile

Kopieren

using System;


class Program

{

  static void Main(string[] args)

  {

    // mehrere Befehle

    Console.Write("Beispiel Text.");

    Console.Write("Dieser Text steht in der gleichen Zeile.");

    Console.Write("Der Text hier auch...");

  }

}

Beispiel: Text in mehreren Zeilen

Kopieren

using System;


class Program

{

  static void Main(string[] args)

  {

    // mehrere Befehle

    Console.WriteLine("Beispiel Text.");

    Console.WriteLine("Dieser Text steht in der gleichen Zeile.");

    Console.WriteLine("Der Text hier auch...");

  }

}

Konsolen-Output

Beispiel Text.

Dieser Text steht in der gleichen Zeile.

Der Text hier auch...

 

leere zeile

Mit Console.WriteLine() gibt man Text auf der Konsole aus und springt gleichzeitig in die nächste Zeile. Bei einer weiteren Konsolenausgabe wird dann in der neuen Zeile weitergeschrieben. Daher erscheint in der Ausgabe oben eine zusätzliche leere Zeile, die durch das dritte Console.WriteLine() im Code-Beispiel verursacht wird.

Programm schließt sich

Damit sich die Anwendung nach der Ausführung nicht sofort schließt, muss ein weiterer Befehl verwendet werden. Mit Console.ReadLine() kann der Nutzer dazu aufgefordert werden, etwas in die Konsole zu schreiben. Die Konsole wird erst geschlossen, sobald Enter gedrückt wird, um den eingegebenen Text zu "senden".
