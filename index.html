Decode encrypted unicode Text

Decode text that is encrypted in Minecraft resource and behavior packs.

How does it work?

1) Copy the unicode text from the text file, the unicode text should look like this
2) Paste the text inside the text area above and click on "Decode Text"
3) Click on "Copy Text" and replace the text in your text file with the new one

Code Example of unicode encrypted text:

{
  "\u0066\u006F\u0072\u006D\u0061\u0074\u005F\u0076\u0065\u0072\u0073\u0069\u006F\u006E": "\u0031\u002E\u0031",
  "\u0061\u006E\u0069\u006D\u0061\u0074\u0069\u006F\u006E\u002E\u0074\u0065\u0073\u0074": {
    "\u0063\u006F\u006E\u0074\u0072\u006F\u006C\u006C\u0065\u0072\u002E\u0074\u0065\u0073\u0074",
    "\u0064\u0065\u0066\u0061\u0075\u006C\u0074": {
      "\u0074\u0065\u0073\u0074\u005F\u0074\u0065\u0073\u0074"
    }
  }
}

If the text file looks like this or the text gets displayed as "\u0066" for example, then you can easily decode this text on that webpage.
